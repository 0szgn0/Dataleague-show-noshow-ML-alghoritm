# 🏥 Dataleague: Show-NoShow ML Algorithm

Hi there! 👋 This repository contains my machine learning solution for the **DataLeague** competition. The main objective of this project was to predict whether a patient will show up for their medical appointment or not (Show / No-Show). 

As a statistics student, this was an excellent hands-on project to apply and improve my data science skills!

## 🎯 Project Overview
Predicting appointment no-shows is a real-world problem that helps healthcare providers optimize their schedules. In this project, I performed data exploration, engineered features, and built machine learning models to make the most accurate predictions possible.

**🏆 Final Competition Score:** I achieved a Private Score of **0.5063131** on the final leaderboard.

## 🛠️ Tech Stack & Tools
- **Language:** Python 🐍
- **Data Handling & EDA:** Pandas, NumPy
- **Machine Learning:** Scikit-Learn, LightGBM, CatBoost
- **Environment:** Kaggle Notebook

## 📂 What's in this repository?
- `Dataleague.ipynb`: Contains the full workflow: Exploratory Data Analysis (EDA), feature engineering, model training, and prediction generation.
- `README.md`: Project overview.
